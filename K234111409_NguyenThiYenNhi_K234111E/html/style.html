CSS Style

CƠ PHẠP VỀ CSS STYLE

CSS (Cascading Style Sheets) là ngôn ngữ được sử dụng để định dạng và trang trí giao diện cho các trang web HTML. CSS giúp kiểm soát màu sắc, font chữ, khoảng cách, bố cục và nhiều thuộc tính khác của các phần tử HTML.

CSS có nhiều cách áp dụng vào trang web, mỗi cách có ý nghĩa và ứng dụng riêng biệt. Việc lựa chọn phương pháp phù hợp sẽ giúp code dễ bảo trì và tối ưu hơn.

CÁC LOẠI STYLE VÀ CÁCH SỬ DỤNG

1. Inline Style

Định nghĩa: Inline Style là cách viết CSS trực tiếp trong thuộc tính style của thẻ HTML.

Cú pháp:
<thẻ style="property: value; property: value;"> Nội dung </thẻ>
Ví dụ:
<p style="color: red; font-size: 20px;"> Đây là đoạn văn có màu đỏ </p> <div style="background: blue; padding: 10px;"> Nội dung </div>
Ưu điểm
  • Áp dụng nhanh chóng
  • Ưu tiên cao nhất
  • Phù hợp cho thay đổi nhỏ
Nhược điểm
  • Khó bảo trì
  • Không tái sử dụng
  • Code dài, rối

2. Internal Style

Định nghĩa: Internal Style là cách viết CSS trong thẻ <style> ở phần <head> của file HTML.

Cú pháp:
<head> <style> selector { property: value; property: value; } </style> </head>
Ví dụ:
<head> <style> h1 { color: blue; text-align: center; } .container { background: #f0f0f0; padding: 20px; } </style> </head>
Ưu điểm
  • Áp dụng cho toàn trang
  • Dễ quản lý hơn inline
  • Phù hợp trang đơn
Nhược điểm
  • Không dùng lại được
  • Tăng kích thước HTML
  • Khó bảo trì nhiều trang

3. External Style

Định nghĩa: External Style là cách viết CSS trong file riêng (.css) và liên kết vào HTML thông qua thẻ <link>.

Cú pháp:
<!-- Trong file HTML --> <head> <link rel="stylesheet" href="style.css"> </head> /* Trong file style.css */ selector { property: value; }
Ví dụ:
<!-- HTML --> <head> <link rel="stylesheet" href="mainstyle.css"> </head> /* File mainstyle.css */ body { font-family: Arial, sans-serif; background: #f0d2de; } .button { background: #954C95; color: white; padding: 10px 20px; }
Ưu điểm
  • Tái sử dụng nhiều trang
  • Dễ bảo trì, quản lý
  • Giảm kích thước HTML
  • Cache được
Nhược điểm
  • Cần request thêm file
  • Phải quản lý nhiều file

KHUYẾN NGHỊ SỬ DỤNG

Nên sử dụng:

  • ✓ External Style: Cho hầu hết các dự án web, đặc biệt là website có nhiều trang
  • ✓ Internal Style: Cho trang đơn, landing page, hoặc khi test nhanh
  • ⚠ Inline Style: Chỉ khi cần override style nhanh hoặc style động bằng JavaScript

Lời khuyên:

Trong thực tế phát triển web chuyên nghiệp, External Style là lựa chọn tốt nhất vì dễ bảo trì, có thể tái sử dụng và tối ưu hiệu suất. Bạn nên tổ chức CSS thành các file riêng biệt theo chức năng (ví dụ: layout.css, components.css, utilities.css) để dễ quản lý hơn.
