TỔNG QUAN VỀ HTML

Cấu trúc tổng quan của một HTML

HTML (HyperText Markup Language) là ngôn ngữ đánh dấu được sử dụng để tạo và cấu trúc nội dung trên web. Một tài liệu HTML cơ bản bao gồm các thành phần chính sau:

<!DOCTYPE html> <html lang="vi"> <head> <meta charset="UTF-8"> <title>Tiêu đề trang</title> </head> <body> <!-- Nội dung trang web --> </body> </html>

Các thành phần của một thẻ (tag)

Một thẻ HTML bao gồm các thành phần cơ bản:

  • Thẻ mở (Opening tag): <tên_thẻ> - Đánh dấu điểm bắt đầu
  • Nội dung (Content): Nội dung bên trong thẻ
  • Thẻ đóng (Closing tag): </tên_thẻ> - Đánh dấu điểm kết thúc
  • Thuộc tính (Attributes): Cung cấp thông tin bổ sung cho thẻ

Các thẻ HTML thường dùng

<h1> - <h6>

Thẻ tiêu đề, từ cấp 1 đến cấp 6

<p>

Thẻ đoạn văn bản

<a>

Thẻ liên kết (hyperlink)

<img>

Thẻ chèn hình ảnh

<div>

Thẻ container, nhóm các phần tử

<span>

Thẻ inline, nhóm văn bản

<ul> / <ol>

Danh sách không thứ tự / có thứ tự

<li>

Phần tử trong danh sách

<table>

Thẻ tạo bảng

<form>

Thẻ tạo biểu mẫu

<input>

Thẻ nhập liệu

<button>

Thẻ nút bấm

JAVASCRIPT

Ý nghĩa của Javascript

JavaScript là ngôn ngữ lập trình kịch bản (scripting language) được sử dụng để tạo và kiểm soát nội dung động trên trang web. JavaScript giúp trang web trở nên tương tác, phản hồi với người dùng, xử lý sự kiện, thao tác với DOM, và nhiều tính năng khác.

Cách sử dụng Internal và External Javascript

Internal Javascript

Viết mã JavaScript trực tiếp trong file HTML, bên trong thẻ <script>:

<script> function sayHello() { alert("Xin chào!"); } </script>
  • Ưu điểm: Đơn giản, không cần file riêng
  • Nhược điểm: Khó bảo trì, không tái sử dụng

External Javascript

Viết mã JavaScript trong file riêng (.js) và liên kết vào HTML:

<!-- Trong file HTML --> <script src="script.js"></script> /* Trong file script.js */ function sayHello() { alert("Xin chào!"); }
  • Ưu điểm: Dễ bảo trì, tái sử dụng được
  • Nhược điểm: Cần quản lý nhiều file

Ví dụ thực tế

Internal Javascript:

<button onclick="showMessage()">Click me</button> <script> function showMessage() { document.getElementById('result').innerHTML = 'Bạn đã click!'; } </script>

External Javascript:

<!-- HTML --> <button onclick="showMessage()">Click me</button> <script src="main.js"></script> // File main.js function showMessage() { document.getElementById('result').innerHTML = 'Bạn đã click!'; }
